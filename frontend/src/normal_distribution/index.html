Galton Board & Central Limit Theorem

This simulation demonstrates how random events create a normal distribution, mirroring how financial returns often distribute.

Key Concepts:

  • Central Limit Theorem (CLT): The sum of many independent random variables tends toward a normal distribution
  • Each peg collision represents a random market movement
  • Final ball position shows cumulative effect of many small random events
  • Most balls cluster near center (average return)
  • Fewer balls reach extremes (rare events)

Financial Interpretation:

μ (mu) = Average return (center cluster)

σ (sigma) = Volatility (spread of balls)
